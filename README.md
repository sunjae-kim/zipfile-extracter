# Test Automation

테스트를 자동화 해보자.

사용한 python 버전: 3.7.3

## Usage

빈 디렉터리에 평가 폴더와 main.py 파일을 넣는다.

```
평가.zip
main.py
```

파일을 실행한다.

```bash
python main.py
```
